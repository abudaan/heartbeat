The recording starts at the position of the playhead. Preroll is on by default, which means that you can record during the precount. This is handy for upbeats. You can not (yet) record events before the first bar.

To enable Web MIDI API on Chromium, Chrome or Canary:
- point your browser to chrome://flags
- search for MIDI and click 'enable'
- restart your browser
